# QR Code Generator Web App

A modern, interactive web application to generate QR codes with live preview, custom colors, and optional brand logo integration. Users can generate QR codes instantly without page reload, download them, and customize the appearance.

---

## Features

- Generate QR codes from any URL instantly
- Live preview of QR code without page reload
- Add a custom logo to the center of the QR code
- Remove logo dynamically
- Customize QR code color
- Download QR codes as PNG
- Minimal, clean, and responsive dark-themed UI
